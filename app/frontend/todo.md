# TrackAjo Waitlist Landing Page - Development Plan

## Design Guidelines

### Design References
- Clean fintech landing pages (Piggyvest, Cowrywise)
- Trust-focused, minimal design

### Color Palette
- Primary: #0A2463 (Deep Trust Blue - background)
- Secondary: #1B3A8C (Medium Blue - sections)
- Accent: #3E92CC (Light Blue - buttons, highlights)
- Hover: #2D7BB5 (Button hover)
- White: #FFFFFF (Text, cards)
- Light Gray: #F0F4F8 (Secondary text)
- Success Green: #28A745 (Confirmation message)

### Typography
- Font: 'Segoe UI', system-ui, sans-serif
- Headings: Bold, white
- Body: Regular, white/light gray

### Key Component Styles
- Signup form: White card on blue background, rounded corners, subtle shadow
- Buttons: Accent blue with white text, rounded
- Input fields: Clean borders, good padding

## Files to Create

1. **index.html** - Main landing page with React via CDN, Bootstrap via CDN
2. **style.css** - Custom styles (blue theme, layout)
3. **script.js** - React components (landing page, signup form, confirmation)
4. **server/server.js** - Express backend server
5. **server/models/Waitlist.js** - Mongoose model
6. **server/package.json** - Backend dependencies
